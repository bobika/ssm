2.5.1.2算法接口描述

算法调用接口：

public double[][] ClusteringbyKMeans(double[][] Data,int Row,int Col,int Classes, int DistanceType,int MaxIteration,int InitType,int Normalization);

Classes 的值仅仅是我们希望从数据集中得到的组、簇或类的数量。

DistanceType 的值表示数据点与点集中心的距离的计算方法。提供了三种对象间距离测量的方法:

1：曼哈顿距离: 基于水平和 / 或垂直路径的网格上两点之间的距离，换句话说，沿着网格线进行计算。它也被称为城市街区距离。

2：欧几里得距离: 两点之间最常见的距离度量。这是唯一最短的路。

3：闵可夫斯基距离: 欧几里得距离和曼哈顿距离的结合。

MaxIteration 最大迭代次数提供了对算法处理的一些控制，以防止出现可能非常长的过程。

InitType 表示初始化类中心的方法。提供了两种初始化方法：

1：前 K 个记录被用作初始的集群中心。这种方法需要谨慎处理，以防数据被打乱。

2：随机替换。K 簇初始中心的记录是随机选取的。

Normalization 表示数据归一化方法。有三种数据归一化方法：

1：数据保持原样。

2：对于每一点，归一化值为，其中。换句话说，每个值的绝对值除以变量或数据列中所有绝对值的总和。

3：对于每个变量C，找出C的最小值和最大值，然后是 C[i] =(C[i]-min)/(max-min)。换句话说，在O和1之间重新缩放数据，其中C的最小值为0。最大值是1。
